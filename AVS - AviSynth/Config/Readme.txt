AVISynth syntax highlighting 
files for Notepad++

Usage:
1) Go to "Language"->"Define your language..."
2) Click "Import..." in window
3) Pick "AVISynth syntax.xml" from this archive

To use function autocompletion feature
place "AVISynth autocompletion.xml"
into <NPP_dir>\plugins\APIs folder
and rename file to "AVISynth.xml".
Read Notepad++ documentation for
details.
