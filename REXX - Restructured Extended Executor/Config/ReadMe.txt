 Open Object Rexx / NetRexx Erweiterung für Notepad++
 ====================================================
 
 Inhalt:
 
 Rexx.api            --> API zur Autovervollständigung
 Rexx.xml            --> API zur Autovervollständigung mit Funktionsparameter-Einblendung
                         basierend auf Open Object Rexx Version 4.01 (Mai 2010)
 userDefineLang.xml  --> Syntax-Highlighting für Notepad++
 makeXML.rex         --> Rexx-Script zur Erstellung der Rexx.xml aus der Rexx.api
 ReadMe.txt          --> Diese Datei
 
 Installation:
 
 Kopiere die Datei "userDefineLang.xml" in das Hauptverzeichnis von Notepad++.
 Kopiere die Dateien "Rexx.api" und "Rexx.xml" in das Unterverzeichnis "Notepad++\plugins\APIs".
 
 Hinweis:
 
 Sollte sich bereits eine Datei namens "userDefineLang.xml" im Notepad++
 Verzeichnis befinden, so sollte sie vorher gesichert werden oder einfach nur
 um den Rexx-Sprachanteil ergänzt werden.

 Open Object Rexx / NetRexx Extension for Notepad++
 ==================================================
 
 Content:
 
 Rexx.api            --> API for autocompletion
 Rexx.xml            --> API for autocompletion with displaying of the function parameter
                         basing on Open Object Rexx Version 4.01 (May 2010)
 userDefineLang.xml  --> Syntax-Highlighting for Notepad++
 makeXML.rex         --> Rexx-Script to create the Rexx.xml from the Rexx.api
 ReadMe.txt          --> This file
 
 Copy the file "userDefineLang.xml" in the main directory of Notepad++.
 Copy the files "Rexx.api" and "Rexx.xml" in the subdirectory "Notepad++\plugins\APIs".
 
 Hint:
 
 If there is already a file named "userDefineLang.xml" in the Notepad++ directory
 then you better save it first or simply add the part of the Rexx language to
 that file.
