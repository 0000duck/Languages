WSH Panel Mod syntax highlighting and autocompletion for Notepad++

-------------Description----------------------------------------------------------------------------------------------
WSH Panel Mod is one of the component for foobar2000 and can be used
 in JScript and VBSript containing the custom functions, methods, and properties. 
This files will be possible to highlighting and autocompletion almost all the
custom functions and methods by Notepad++.
And some functions and methods, the description is displayed by tooltip, 
but these description reprinting WSH Panel Mod's documentation.


-------------Usage----------------------------------------------------------------------------------------------
1. In the main menu, go to "Language"->"Define your language..."
2. Click the "Import..." button.
3. Browse to the "jscript-wshpm_udl.xml".
4. Close the dialogue and exit Notepad++.
5. Copy the "jscript-wshpm.xml" into APIs folder (e.g. C:\Program Files\notepad++\plugins\APIs).
6. Launch the Notepad++ again and choose the "JScript-WSHPM" from the "Language".
    If doesn't work autocompletion, check the Auto completion option(Settings -> Preferences -> 
    Auto completion-> Enable Auto-completion on each input) and restart Notepad++.
